平台差异
  1. W
    Web 浏览器：无 directPipe 本地 daemon → 「开始使用」直接进选择器；无扫码（用直连/粘链）。一次性品牌欢迎仍保留（首次访问）。
  2. M
    手机 native：无本地 daemon → 扫码连电脑主机为蓝主 CTA；粘链(描边)/直连(链接)为次。连上后即远程遥控本机代理（relay E2E）。
  3. D
    桌面 Electron：唯一有本地 daemon 的平台 → 走 S1→S2 自动连主路径。
状态
布局
Web=磨砂窗体 + 居中白卡（浏览器顶栏带标签标题）；手机=圆角设备 + 白屏全幅居中（状态栏 9:41）。
hover/活跃

各平台连接中=蓝顶环 spinner；失败走各自错误提示（手机为底部 toast/卡内 Alert）。
深色变体
跟随系统：.dark 令牌（窗体 #0d1117 / 卡片 #161b22 / 主蓝 #1f6feb），结构不变。
