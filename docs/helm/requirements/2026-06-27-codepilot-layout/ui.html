Helm · UI 设计稿 · 主界面布局 + 主题色（参照 CodePilot · 骨架版）

本地需求设计稿（gate-1 · 待董事长看 PNG）。需求 2026-06-27-codepilot-layout：参照开源项目 CodePilot 把 Helm 主界面的布局结构与主题色改版。 本版只做布局骨架 + 主题色，细节标「后续」。核心 = 一条贯穿整宽的统一顶栏 + 主体 4 部分（① 左侧栏 ② 中区画布 ③ 右侧栏 ④ 目录树）+ 3 个独立 toggle（左栏 / 右栏 / 目录树，各自独立、可同时展开、additive 非互斥）。 主题换成 CodePilot 的 「github」主题（内部命名 codePilot） —— GitHub Primer 浅色：冷灰中性 + GitHub 蓝仅点缀，浅 + 深双适配。关键纠正：列表 / 导航选中态 = 很淡的灰（sidebar-accent #eaeef2 + 深灰字 #3d444d + 加粗），绝非黑底白字；primary 蓝（#0969da）只用于主按钮 / 链接 / focus。所有色值按 CodePilot 源码（themes/github.json + ChatListPanel/UnifiedTopBar/card-primitives.tsx）逐项取值。

布局骨架 · 1

主页 · 默认空态（浅色）

连上主机后落地的空态。统一顶栏横跨整宽（透明、融入窗口背景）：左 = 交通灯 + 左栏开关 + 标题「新对话」+ 项目名（弱化）+ ··· 菜单；右 = develop 分支 + 目录树开关 + 右栏开关。 顶栏下方是 4 张浮动卡片同一 y 起：本屏只展开 ① 左栏（Helm 自己的：主机切换器胶囊 + 新对话 + 搜索会话 + 对话树「项目→对话→subagent」 + 设置）+ ② 中区（居中问候「晚上好」+ Composer），③ 右栏与 ④ 目录树收起。 微透 vibrancy 分层：窗口背景层 = macOS 'sidebar' 磨砂材质（顶栏 + gutter 留白直接露这层 → 整窗微透磨砂）；左栏 = white 58% 半透磨砂（带窗口 tint）；中区/右栏/目录树 = 实色白卡 + 1px 边框 #d1d9e0，扁平绝不玻璃。「新对话」是 ghost 行，选中态用很淡的灰。CodePilot 只接管「外壳 + 主题色」，左栏内部是 Helm 对话树。

主题 · 点击效果 / 状态 / 右键
切换
浅 / 深由系统外观或设置里手动切换驱动（后续由设置模块提供开关）。所有页面（首页 / 对话页 / 设置…）共用同一套令牌，统一适配。
主色 primary
GitHub 蓝 #0969da（深 #1f6feb），只点缀：主操作按钮（发送键）、链接、focus 蓝环、品牌强调。绝不铺选中行、不铺大面积。原 Helm 绿 #20744A 在此版退场。
选中态
列表 / 导航激活·选中（新对话当前项 / 当前会话 / 当前导航）= 底 sidebar-accent #eaeef2（很淡的灰） + 字 #3d444d 深灰 + 加粗。绝不是黑底白字、绝不用 primary。hover 同为淡灰底。
状态色
成功绿 / 警告橙 / 错误红 / 信息蓝仅用于语义反馈（运行点 / 脏文件 / 错误条），不进主壳常规底色。
布局
色板屏无设备交互；深色缩略证明浅 / 深双适配。后续：每个页面单独出深色稿由各模块负责。

无。
