本地需求设计稿（gate-1 迭代 · 待回写 canonical）。覆盖 2026-06-25-home-shell：连上主机后落地的主页面主体。 本轮按董事长反馈 A–K 重画，逐像素贴 Codex：删掉横跨整窗的标题栏，改为 三个独立整体——① 左侧栏（全高列，窗口 chrome 细条在左上 = 交通灯 + 侧栏开关 + ‹ › 前进/后退 + 主机切换器胶囊 + 对话树 + 设置）、② 中区画布（自带 canvas 顶栏：标题 / ··· / 打开位置 / 环境信息（置顶摘要） / 右面板开关；侧栏开关与前进/后退不在此，在左栏顶的窗口细条；空态精简——这些上下文控件对话开始后才出现）、③ 右侧栏（自带 tab 头：审查固定 + 已开页签 + 新选项卡 + ⤢放大 / 收起 + 左缘拖宽）。本轮桌面 only；紧凑 / 手机态暂缓，后续由桌面统一适配。

交互注解
点击效果
  1. 1
    审查页签（固定第一）：点它回改动 diff。审查默认不可关（悬浮的 ✕ 置灰，开放项见右键菜单注）。

    新选项卡 +：点开下拉【终端 ⌃⇧G / 浏览器 ⌘T / 文件 ⌘P】，选一个就在 tab 头新增对应类型页签（见 S5）；快捷键同等触发。
  3. 3
    diff 行操作：悬浮行尾出「+ 行评论」；可按块 / 文件 Stage 或 Revert；标题汇总增删行数。
  4. 4
    ⤢ 放大 / 收起：⤢ = 面板放大占满中区 / 还原（见 S13）；其右的收起把整列收起、开关回到 canvas 顶栏右（位置不变，见 S3）。放大 / 缩小只在右面板内提供。
  5. 5
    页签关闭 ✕（C②）：页签悬浮时行尾浮出 ✕，点它关该页签。关浏览器 / 文件 / 终端 = 仅布局（可重开）。审查不可关。
  6. 6
    左缘拖拽改宽（C④）：拖右栏左缘手柄改面板宽度，每工作区记忆（手柄态见 S12）。
  7. +
    页签拖拽调序（C③）：页签可左右拖拽调序，拖动时显蓝绿放置指示线（本屏「浏览器」后那条竖线即指示线）。

    启动器（打开默认态 · 反馈 ②）：右面板刚打开、还没选内容时显启动器列表（审查 ⌃⇧G / 终端 / 浏览器 ⌘T / 文件 ⌘P）；点某行打开对应 tab；详见下方「启动器 · 流转」。

左栏 · 对话树（项目 → 对话 → subagent 无限嵌套）
