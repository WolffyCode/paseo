Helm · 多模型对决 Agent Race（gate-1 提案）

创新功能设计稿（gate-1）。一个 prompt 一键派给 N 个参赛者（每个 = 提供方·中转站·模型 三层选型 + 模式），各自在隔离 worktree 并行跑，完成后并排对比、选「赢家」一键收尾。本轮 桌面 only，浅色主题，accent #20744A。每屏含完整交互注解（布局/默认/hover/活跃/运行/禁用/空/加载/错误/离线/右键/点击效果）。
