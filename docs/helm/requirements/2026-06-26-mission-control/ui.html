「队列已清空」小态
触发
有代理但「需要你」为 0：MC1/MC2 队列区显居中「全部处理完毕 ✓ · 没有需要你的事」，分桶/树照常。
状态
布局
桌面：四格各为指挥舱主区的一种边界态（空/加载/离线/错误）。紧凑/手机：本轮不做。
空
无代理：bot 图标 + 标题 + 引导 + 新建代理（主）/ 连接主机。daemon 已连，只是没代理。
加载
分桶/队列骨架 shimmer + 顶栏 spinner「正在加载舰队…」；典型 <1s。
离线
顶部 amber 横幅说明哪台离线 + 影响 + 重连；离线主机行 dashed + 重试进度 spinner；在线主机照常。
错误
红 alert 图标 + 「无法加载」+ 失联原因 + 自动重试计数 + 重试（主）/ 查看诊断。
默认/hover/active
按钮同全局规则（pri 绿 / out 中性 / hover 加深）；横幅「重新连接」hover 下划线。
运行/禁用
离线主机重连中 = spinner；加载骨架不可点；空态/错误态按钮恒可点。
