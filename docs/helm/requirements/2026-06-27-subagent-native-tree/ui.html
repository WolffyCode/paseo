布局

  • 桌面：左栏全高树，节点按层缩进；深层超 8 层后缩进封顶，仅靠 chevron/图标区分。
  • 紧凑 / 手机：同结构，左栏为覆盖式抽屉；缩进步长更小。

状态

  • 默认：节点 = 图标 + 标题（任务描述，缺则 agent 类型，再缺则「Sub-agent」）+ 状态点。
  • hover：行底色变亮（web only；native/紧凑常显操作）。
  • active / 选中：蓝底；选中态由路由派生（不另存）。
  • 运行：脉冲绿点（节点出现即 running）。
  • 完成：idle 灰点。出错：红点。
  • 可展开：有后代 → chevron ▾/▸ + 后代计数角标；无后代 → 无 chevron。
  • 加载（树重建中）：daemon 重启 / 首次扫描时，节点逐个补入（近实时，非整屏 spinner）。
  • 空：无 subagent 的对话不显任何子行（不占位）。
  • 只读标记：provider 内部子带 🔒；paseo-run 子无 🔒。
  • 能力门关：旧主机 → 只显直接子层，更深不出现，树尾给升级提示（见 S3）。

右键菜单

  • provider 内部子（🔒 只读）：① 展开/收起 ② 复制目录 ③ 在树中定位父；禁用/不显：发消息、独立重命名、独立归档、派生到本地（与现有 observed 一致）。
  • paseo-run 子：保持现有可控右键（重命名 / 归档(级联) / 拆分 / 关页签 / 打开位置…），不变。
  • 主对话（根）：归档时级联把整 observed 子树标归档（复用 archive-subagent），其子树转为只读历史。

点击效果

  • 点 chevron：仅展开/收起该节点子树（复用折叠 store）。
  • 点节点行：导航到该 subagent（navigateToAgent）→ 打开其对话（S2）。
  • 点状态点 / 角标：无独立动作（纯指示）。
  • 新节点出现：任务跑动中，子/孙近实时（秒级）插入到正确父之下；状态点随生命周期变化——不闪、不挂错父（架构 §4·1/§4·2 保证）。

🔒 这是 Claude 内部子任务，只读——由 provider 编排，不能发送 / 中断。

布局

  • 桌面：可在右面板 tab 或中区打开（复用既有对话承载）；顶部节点标题 + 🔒 + 状态点。
  • 紧凑 / 手机：全屏对话；底部为只读提示条替代输入框。

状态

  • 加载：拉取该节点完整 timeline 中（骨架占位）。
  • 正常（只读时间线）：消息 / 工具调用 / 思考，与 provider 原生记录一致、无缺漏、无重复（架构单一 writer + handoff replace 保证）。
  • 增长中：节点仍 running → 内容近实时（~200ms 级）向下追加；handoff（父 turn 完成）时做一次权威 REPLACE（经 timeline epoch 重置，不闪/不并列重复）。
  • 空：节点刚出现、尚无消息 → 占位「等待该子任务输出…」。
  • 错误：仅该节点落盘读失败 → 视图内错误态（见下框），不连累整棵树。
  • 离线：主机断连 → 复用既有离线态；重连后恢复。
  • 只读提示条：始终替代输入框（provider 内部子）。
错误态示意：⚠ 无法读取该子任务对话（落盘缺失或损坏）。其它节点不受影响；可重试。

右键 / 顶部菜单

  • 只读视图无发送框、无中断按钮；顶部「···」仅：复制深链接 / 在树中定位 / 复制目录（写操作项禁用）。

点击效果

  • 滚动到底：running 节点保持贴底跟随新内容；用户上滚则停跟随（复用既有 timeline 行为）。
  • 点工具调用：展开/收起其详情（复用既有 timeline item 交互）。
  • paseo-run 子（对照）：同一视图但有输入框、可中断（observed=false），无 🔒、无只读条。

⤴ 升级主机以查看更深层 subagent

旧 daemon 无「原生 subagent 树」能力 → 仅直接子（live）可见，更深层不拉取；提示挂在该处。

布局

  • 提示为轻量 inline pill，挂在「能拿到的最深层」之下，不弹模态、不打断。

状态

  • 能力门开（新主机）：无此提示；深层节点正常出现（S1）。
  • 能力门关（旧主机）：只显直接子层 + 此提示。不降级模拟（不在客户端拼造深层）、不崩溃、不假装显示。
  • 检测来源：单点纯 selector 读主机能力 flag（架构 §4·10）；下游读一个干净布尔，无散落兼容分支。

右键菜单

  • 无（pill 仅一个动作）。

点击效果

  • 点 pill：打开主机升级指引（复用既有「升级主机」入口）；不改树本身。
