点击效果
  1. 1
    停止 / 启动：运行时点「停止」→ 弹确认「这会立即停止内置 daemon。连接到它的运行中 agents 和 terminals 会被停止。」→「暂停并停止」；确认后状态点转灰、文字「未运行」、按钮变绿「启动」。停止态点「启动」→ 立即拉起，状态点转琥珀「启动中…」→「运行中」。
  2. 2
    随 Helm 启动（开关）：开 = Helm 托管并在打开 App 时自动运行内置 daemon；关 → 同「停止」确认（暂停并停止），之后不再自动起。即时持久化（设备本地）。
  3. 3
    退出后保持 daemon 运行（开关）：开 = 关闭 Helm 后 daemon 后台续跑（agent 不中断）；关 = 退出即随之停止。即时持久化。
  4. 4
    复制路径：复制 daemon.log 路径到剪贴板，提示「日志路径已复制」。无日志路径时此按钮不显示。
  5. 5
    打开日志：弹「Daemon 日志」对话框（路径 + 文件内容，等宽可选中复制；空文件显「（日志文件为空）」）。无日志时按钮禁用。
  6. 6
    查看状态：运行 paseo daemon status，弹「Daemon 状态」对话框（等宽输出 + 关闭 / 复制）；运行中按钮显「加载中」禁用；失败时对话框显「获取 daemon 状态失败：…」。
  7. 7
    高级设置 ↗：浏览器打开 paseo.sh/docs/configuration。
