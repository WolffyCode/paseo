Helm · UI 设计稿 · 壳子（纯骨架）

本地需求设计稿（gate-1 复审 · 待董事长看 PNG）。需求 2026-06-28-shell：把 Helm 主壳重写为「统一顶栏 + 主体」上下布局，主体承载两种页面：① 对话页（左栏 + 中区 + 右栏 + 目录树）② 设置页（左导航 + 内容）。 gate-1 反馈：壳子东西太多。本版按反馈收窄成纯骨架——壳子 = 只有布局规则 + 全态交互 + 对应按钮图标。各内容区一律换成虚线空占位 + 区域名；删掉对话树 / 搜索 / 项目 / host 切换器 / 右键菜单 / 消息 / Composer / 右栏 tab 内容 / 目录树文件 / 设置导航项 / 面包屑 / ··· 菜单 / 分支标签。 配色 / 圆角 / 几何 / hover 特效原样保留 codePilot 浅色 GitHub（github.json）主题。桌面 only；移动 / 紧凑态 deferred。

对话页 · 1

对话页 · 骨架默认态（壳基线 · 浅色）
