怎么用这份文档

每个需求按 README 工作流：读现状 → 出本地一对 → 回写本文件。
新增一屏时，复制下方「屏模板」，把视觉画进 .canvas，并把「交互注解块」每个维度填满（确实没有就写「无」）。两份 canonical 强制同步：这里画 UI，product.md 同步写文字。
